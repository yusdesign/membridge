# Contributing to MemBridge

☘️ Thank you for helping grow this garden of memory!

## Ways to Contribute

- 🐛 **Report bugs** - Open an issue
- 💡 **Suggest features** - Start a discussion
- 📚 **Improve docs** - Submit a PR
- 🎨 **Design** - Logo, assets, themes
- 🌍 **Translate** - Make MemBridge multilingual

## Development Setup

```bash
git clone https://github.com/yusdesign/membridge.git
cd membridge
# That's it! No dependencies needed.
```

Testing

```bash
python mempalace-bridge.py status
python -m pytest tests/
```

Code Style

· Pure Python 3.6+ compatible
· No external dependencies
· PEP 8 where possible
